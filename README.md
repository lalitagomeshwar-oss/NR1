# My AI Platform

A world-class AI study dashboard with:
- Student dashboard
- AI assistant chat
- Voice-to-voice support
- Voice-to-video lecture mode
- Chapter-based learning
- Whiteboard-style explanations

## Features
- Smooth dashboard UI
- Dynamic chapter parsing
- Voice-ready architecture
- Modular Next.js project structure

## Tech Stack
- Next.js
- TypeScript
- React
- Vercel

## Project Structure
- app/ → pages and layouts
- components/ → reusable UI
- utils/ → helper functions
- types.ts → shared types
- public/ → images and assets

## How to Run
1. Install dependencies
2. Start development server
3. Open the app in browser

## Goal
Build a smooth, modern, AI-powered learning platform for students.
